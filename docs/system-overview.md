# System Overview: UBE-HR

UBE-HR is a modular, full-stack Human Resources application. It features a Node.js/Express backend and a React frontend, both organized using strict feature-based architecture. The system enforces best practices for security, type safety, testing, and documentation, and leverages technologies such as Prisma, Zod, Zustand, and Tailwind CSS. All planning, implementation, and documentation should align with these principles and the project’s technology stack.

- **Backend:** Node.js, Express, TypeScript, Prisma, Zod, PostgreSQL
- **Frontend:** React, TypeScript, Vite, Zustand, Zod, Tailwind CSS
- **Architecture:** Modular (feature-based), strict separation of concerns
- **Security:** Environment variables for secrets, validation everywhere
- **Quality:** Comprehensive testing, documentation, and code review

Refer to this document for context when planning or implementing any feature in the project.
